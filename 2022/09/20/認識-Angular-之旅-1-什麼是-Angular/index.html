認識 Angular 之旅 (1) - 什麼是 Angular

發表於2022-09-20|更新於2022-12-08|Angular

什麼是 Angular？

Angular 是由 Google 所開發基於 TypeScript 所編寫的現在流行三大框架之一，它強大的地方在於整合了許多功能給開發者使用，所以也有人將之稱為開發平台。


Angular 與 AngularJS

Angular 原本稱作為 Angular 2，但因為 Angular 2 是重新打造的新一代框架，與之前版本差異甚大，為了區分兩者的差異，1.x 版本稱為 AngularJS，而在這之後的 2.x、3.x、4.x…… 到最新版本都稱為 Angular。
現在 Angular 每半年會穩定更新一個版本，並且不會像 Angular 1.x 到 Angular 2 之間會差異過大，每個版本都可以透過 Angular CLI 的指令無痛升級上去。
