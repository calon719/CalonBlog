認識 Angular 之旅 - 前言

發表於2022-09-20|更新於2022-10-03|Angular

開始學習 Angular 原因

之前在上六角學院的 Vue 直播班時，因為希望透過回答同學的問題來找出並加強自己不懂的地方，所以時常幫同學做 code review，但在做 code review 時常需要花蠻多時間在確認資料的型別，所以當時在找有沒有方法可以快速解決確認資料型別問題，而開始學習 TypeScript，並且在這時後也開始想接觸其他框架，從而接觸到 Angular，並被它的優點吸引住所以開始決定學習 Angular。

認識 Angular 之旅是紀錄我學習 Angular 的過程筆記，主要是透過 Udemy 上 Will 保哥 的 Angular 開發實戰：從零開始以及 Angular 中文文件來學習。
