安裝 Node.js

在安裝 Angular 之前我們需要先安裝 Node.js，我們會需要使用它的 npm 套件管理工具來安裝我們的 Angular CLI（Command Line Interface），而 Node.js 只要到Node.js 官方網站下載，裡面會有穩定版本以及最新版本可以下載。

輸入後會跑出以下文字問你的專案是否要加入 Angular routing，在這邊我們先不要加入所以輸入 N：

再來還會出現一段文字，意思是你要使用哪種工具撰寫 CSS，這邊選擇自己常用的即可：

而以上設定我們可以一開始就以參數的方式加在 ng new 指令的後面：

  • n 為 new 的縮寫
  • --no-routing 是不要加入 Angular routing，要加入的話只要改成 --routing 即可
  • --style=css 是撰寫 CSS 的方式，在這邊是使用原生 CSS，如果是使用 SCSS 則是 --style=scss，其他以此類推

安裝時會順便透過 NPM 安裝套件，所以會需要一些時間，出現以下文字就代表安裝成功了：

第二行的 Successfully initialized git. 是 Angular CLI 會在最後利用 Git 來進行版本控制，並提交第 1 次 commit。

  • --open 參數是啟動開發伺服器後自動開啟網頁，可以簡寫成 -o
