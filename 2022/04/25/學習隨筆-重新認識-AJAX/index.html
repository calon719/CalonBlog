學習隨筆 - 重新認識 AJAX

發表於2022-04-25|更新於2022-04-25|JavaScript

每次見到 AJAX 這個技術名詞，即使知道它的全名 Asynchronous JavaScript and XML 以及是為了提昇使用者體驗，在向伺服器傳送請求時不會造成換頁的技術，但為什麼名稱是用 JavaScript、XML，在最近閱讀到這篇：[FE102] 前端必備：網頁與伺服器的溝通之後終於恍然大悟。

名稱裡面有 JavaScript 是因為透過 JavaScript 去向伺服器傳送請求，而 XML 則是因為早期的格式是使用 XML。

這下以後看到 AJAX 時不會再困惑啦～
